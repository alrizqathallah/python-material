# PENGENALAN PYTHON

## APA ITU PYTHON?

Python adalah bahasa pemrograman yang dibuat oleh **Guido Van Rossum** dan dirilis pada 1991.

Digunakan untuk:

- Pengembangan Web (server-side),
- Pengembangan Aplikasi,
- Matematika,
- Scripting Sistem.

### Apa yang bisa Python lakukan?

- Python dapat digunakan pada sebuah server untuk membuat aplikasi web.
- Python dapat digunakan bersama perangkat lunak untuk membuat alur kerja.
- Python dapat dihubungkan dengan sistem berbasis data.
- Python juga dapat digunakan untuk mengubah data dan membaca berkas.
- Python dapat digunakan untuk keperluan Big Data dan pengolahan Matematika yang rumit.
- Python dapat dimaksimalkan untuk membuat protyping dengan cepat, untuk pengembangan perangkat lunak yang siap produksi.

### Kenapa Python?

- Python dapat digunakan diberbagai sistem operasi (Windows, Mac, Linux, RaspberryPI, dll).
- Memiliki Sintaks dasar yang simpel.
- Dengan Python, programmer dapat menuliskan kode program seminimal mungkin jika dibandingkan dengan bahasa pemrograman lain.
- Kode Python dijalankan menggunakan sistem interpreter, ini memungkinkan kode untuk secepat mungkin di eksekusi. Ini berarti prototip dapat dilakukan secepat mungkin.
- Python dapat dijalankan secara prosedural, berorientasi-objek atau fungsional, tergantung pada kebutuhan.

### Perlu Diketahui

- Versi terbaru dari Python saat in adalah versi 3, ketika tutorial pembelajaran ini dilakukan. Meski demikian Python 2 cukup populer untuk digunakan.
- Penulisan kode Python pada pembelajaran ini akan menggunakan text-editor, atau Integrated Development Environment (IDE).

### Perbandingan Sintaks Python dengan Bahasa Pemrograman lain.

- Bahasa Python didesain dengan tingkat readibility yang tinggi. Hampir mirip dengan penggunaan bahasa inggris pada umumnya degan sedikit sentuhan matematika.
- Python menggunakan baris baru untuk melengkapi sebuah perintah, berbeda dengan bahasa pemrograman lain yang sering menggunakan titik koma atau tanda kurung.
- Python bergantung pada indentasi, menggunakan spasi untuk menentukan cakupan, seperti loop, function dan class. Berbeda dengan bahasa pemrograman lain yang lebih banyak menggunakan {} block.
